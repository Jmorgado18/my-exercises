Exercício 1: Contador Simples
Objetivo: Praticar a impressão de valores e o uso de variáveis.

Crie um programa que imprima números de 1 a 5 no console, um por linha.
Exercício 2: Calculadora de Soma
Objetivo: Trabalhar com variáveis e expressões matemáticas.

Escreva um programa que solicite ao usuário dois números e imprima a soma deles no console.
Exercício 3: Mensagem Personalizada
Objetivo: Manipular strings e usar variáveis.

Crie um programa que peça ao usuário seu nome e idade, e então imprima uma mensagem no console: "Olá [nome], você tem [idade] anos!"
Exercício 4: Palíndromo
Objetivo: Trabalhar com métodos de strings.

Escreva uma função que receba uma string e verifique se é um palíndromo (ou seja, se a string é igual quando lida de trás para frente). A função deve retornar true ou false.
Exercício 5: Checagem de Idade
Objetivo: Usar condicionais e booleanos.

Crie um programa que peça ao usuário sua idade e verifique se ele é maior de idade (18 anos ou mais). Imprima uma mensagem no console indicando se o usuário é maior de idade ou não.
Exercício 6: Jogo de Adivinhação
Objetivo: Utilizar loops e condicionais.

Desenvolva um jogo simples onde o computador escolhe um número aleatório entre 1 e 10 e o usuário tem 3 tentativas para adivinhar o número. Após cada tentativa, informe ao usuário se o palpite foi correto ou não.
Exercício 7: Tabela de Multiplicação
Objetivo: Trabalhar com loops e arrays.

Crie um programa que gere uma tabela de multiplicação de 1 a 10. Utilize um array para armazenar os resultados e um loop for para imprimir a tabela no console.
Exercício 8: Lista de Compras
Objetivo: Usar arrays e loops.

Desenvolva um programa que permita ao usuário adicionar itens a uma lista de compras. O usuário deve ser capaz de adicionar, remover e listar os itens. Use arrays para armazenar os itens e um loop for para listar todos os itens na tela.
Exercício 9: Informações do Usuário com Objetos
Objetivo: Trabalhar com objetos e funções.

Crie um programa que define um objeto usuario com as propriedades nome, idade e email. Escreva uma função exibirInformacoes que imprime essas informações no console em um formato legível.
Exercício 10: Jogo da Forca
Objetivo: Combinar todos os conceitos estudados.

Desenvolva um jogo da forca simples. O jogo deve escolher uma palavra aleatória de uma lista e permitir ao usuário tentar adivinhar a palavra, letra por letra. O programa deve exibir a palavra com as letras corretas reveladas e informar quantas tentativas restam. Use arrays para armazenar a lista de palavras e o estado do jogo, e funções para a lógica do jogo.
